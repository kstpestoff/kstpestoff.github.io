Множества в python содержат элементы множества. Все операции и свойства соответствуют математическим [множествам](https://ru.wikipedia.org/wiki/Множество). 
```python
>>> X = set([1,2,3,4])
>>> Y = {3,4,5,6,7}
#Пересечение
>>> X&Y
{3, 4}
#Объединение
>>> X|Y
{1, 2, 3, 4, 5, 6, 7}
#Разность
>>> X-Y
{1, 2}

```
